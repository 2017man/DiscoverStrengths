# 《发现你的天赋优势》开发规划

## 一、产品与技术定位

| 项目 | 内容 |
|------|------|
| **产品名称** | 发现你的天赋优势 |
| **主 Slogan** | 发现天赋，迈出第一步 |
| **副标（轮换）** | 你的优势，值得被看见 / 天赋是起点，成长是旅程 |
| **国内版技术栈** | 前端：uniapp (H5)；后端：Laravel + dcat-admin；数据库：MySQL 5.7 |
| **支付（v1.0）** | 第三方易支付（微信支付），先跑通流程、验证需求 |

**愿景说明**：我们不仅仅是一个网站——**网站是工具和入口**，最终目标是**帮助更多的人解决困境与迷茫，与用户一起成长**。产品与开发都服务于这一目标，差异化在于有温度的成长陪伴。

## 三、v1.0 开发范围（本期）

### 3.1 功能边界

- **上线**：仅 MBTI 职业性格测试（完整闭环）
- **预留**：首页展示 4 个测评卡片（MBTI、霍兰德、DISC、职业锚），仅 MBTI 可点击，其余 3 个灰色「即将上线」
- **定价**：各测评类型价格在后台（dcat-admin）可配置，前端与支付按配置读取
- **语言**：v1.0 仅中文；双语放在 v1.1

### 3.2 核心闭环

```
用户访问首页 
→ 选择 MBTI（其他 3 个灰） 
→ 测试说明页 
→ 答题（session/localStorage） 
→ 提交 → 计算 → 结果预览（约 30% 免费） 
→ 点击「查看完整报告」 
→ 未付费则易支付 → 回调 → 更新订单与报告状态 → 完整报告 + PDF 下载
```

### 3.3 必须能力

- 测试工具：MBTI 题目展示、答题、计分、结果映射（16 型）
- 报告：免费预览约 30%；付费解锁完整报告；PDF 下载；温馨寄语/成长名言；副标 B/E 轮换；公众号/社群引流
- 用户与支付：匿名答题；H5 微信 openID（可选）；防重复付费（同人同类型）；易支付接入与回调；订单管理
- 后台：测试类型管理、题目与结果配置、**价格配置**、订单与报告查询
- SEO：各页标题/描述/关键词；结构化数据
- 「有温度」：随机暖心文案、结果页共鸣句、错误/空状态暖心话、报告内寄语与名言

## 四、阶段与里程碑（建议）

| 阶段 | 里程碑 | 产出 |
|------|--------|------|
| **M1** | 后端基础 + MBTI 数据与计分 | 数据库表（测试类型、题目、规则、结果码、结果详情）；MBTI 计分与结果码；管理端 CRUD |
| **M2** | 前端 MBTI 全流程（未付费） | 首页 4 卡片（3 灰）；说明页；答题页；结果预览页；报告页（预览 30%） |
| **M3** | 支付与完整报告 | 易支付对接；支付页；回调；订单状态；完整报告解锁；PDF 下载 |
| **M4** | 打磨与上线 | SEO；暖心文案与副标轮换；公众号/社群位；测试与修复 |

具体排期可根据人力与优先级微调；详细任务见《开发计划》。

## 五、数据与扩展原则

- **测试类型独立**：每种测评独立表结构（题目、计分规则、结果码、结果详情），便于后续加霍兰德、DISC、职业锚
- **价格与配置**：测试类型表含价格等配置，后台可改，前端与支付只读配置
- **报告与文案**：成长名言、暖心句、副标等建议配置化或独立表，便于运营修改
